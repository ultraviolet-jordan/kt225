# 225 Emulator Written in Kotlin.

## Requirements
- Kotlin 1.8.20

## Getting Started
- To start the web server using Gradle, you can run the run task:
```shell
./gradlew :http:run
```

- To start the game server using Gradle, you can run the run task:
```shell
./gradlew :game:run
```
